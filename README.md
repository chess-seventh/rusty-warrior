# Rusty Warrior

It's a simple tool to get notifications about my most urgent tasks from Taskwarrior in `rust`.

This project is still a __WIP__ as I am learning `rust`.


## Prerequisites

Have your "copy" of `rust` installed.


## Installation

Simply run:

```bash
cargo build
```

or

```bash
cargo run
```
